# Output Directory

This directory contains locally generated HTML talent card files for development and testing purposes.

**Note:** Files in this directory are automatically ignored by git and not deployed to Heroku.

## File Naming Convention
- `talent-card-{employee_id}.html` - Generated talent card HTML files

## Usage
- **Local Development**: HTML files are saved here when running locally
- **Heroku Production**: This directory is not used (in-memory processing only)
